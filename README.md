# 功能中心

一个集合多种实用功能的在线工具平台。

## 主要功能

- **Pixiv涩图**: 精选优质二次元插画，支持多种筛选，标签搜索
- **硬件排行**: 全网最新的CPU、GPU和手机处理器性能排行榜
- **AI 助手**: 智能AI助手，为您解答各种问题
- **视频播放**: 精选短视频，随机播放
- **买家秀**: 随机淘宝买家秀图片
- **应用搜索**: 海量应用资源，一键搜索下载
